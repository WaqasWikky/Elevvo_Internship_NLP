# Elevvo_Internship_NLP

# NLP Internship Tasks

This repository contains Natural Language Processing projects completed during the internship.

## Tasks

### 1. Sentiment Analysis

Classifies product reviews as positive or negative using TF-IDF and Logistic Regression.

### 2. Fake News Detection

Detects whether news articles are real or fake using text classification techniques.

### 3. Named Entity Recognition (NER)

Extracts entities such as person names, organizations, and locations using spaCy.

### 4. Question Answering

Uses transformer models to answer questions based on a given context.

## Tools & Libraries

* Python
* Pandas
* Scikit-learn
* spaCy
* Transformers
* Google Colab
